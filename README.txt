git status > verifica se há algum arquivo modificado 
git add NOME_DO_ARQUIVO > adiciona arquivos modificados para lista do próximo commit. caractere " . " significa que você adicionará todos os arquivos
git commit -m "INFORMAR MENSAGEM" > realiza um commit
git branch > cria uma nova branch (espécie de workspace)
git checkout NOME_DO_BRANCH > altera branc em que você está trabalhando 
git merge master > unifica codigo
git tag -a INFORMAR VERSÃO -m "INFORMAR MENSAGEM" > versiona o git
git push > atualiza a branch informada
